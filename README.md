# Immigration Pathway Analysis (Canada)

## Overview
This project explores the relationship between study permits and permanent residency (PR) in Canada.

The goal was to understand:
- Which provinces attract the most international students
- Which immigration programs account for most PR approvals
- Whether there are clear patterns in how people transition from study permits to PR

---

## What I Did
- Cleaned and structured two real-world datasets
- Transformed messy data into usable formats using Pandas
- Aggregated and analyzed key metrics
- Built visualizations to highlight trends
- Interpreted results to draw meaningful insights

---

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

---

## Key Findings

### Study Permits
Most study permits are concentrated in a few provinces, especially Quebec and Alberta. This suggests that international students are not evenly distributed across Canada.

### PR Transitions
Permanent residency approvals are dominated by a small number of programs:
- Skilled Worker Program (largest share)
- Sponsored Spouse or Partner
- Canadian Experience Class

Other programs contribute very little in comparison.

### Overall Insight
Even though many students come to Canada, the transition to PR seems to depend heavily on a few major immigration pathways.

---

## Limitations
This analysis is based on aggregated data, so it does not track individual journeys from study permit to PR.
